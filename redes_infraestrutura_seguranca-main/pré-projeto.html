Pré-projeto

O Pré projeto é um projeto em que criamos uma rede, onde conectamos os cabos da internet para conectar aos computadores que no total são em 50 computadores e dividimos em 4 andares. Também colocamos internet no local para os celulares, tablets e notebooks, configuramos o WIFI assim todos tem acesso a internet. Fizemos o RIP para alinhar os IP's.
